AIMET Keras Cross Layer Equalization APIs¶

Introduction¶

AIMET functionality for Keras Cross Layer Equalization supports three techniques:

  • BatchNorm Folding

  • Cross Layer Scaling

  • High Bias Fold

Cross Layer Equalization API¶

The top level Cross Layer Equalization for Keras API is still under development. For the time being, the Primitive APIs can be used to perform each step of Cross Layer Equalization separately.

If the user would like to call the APIs individually, then the following APIs can be used:
